## ğŸ“„ What is FunTraps?

FunTraps is a fully customizable tactical trap system designed for competitive PvP, Factions, Survival SMP, and Minigame servers.

Unlike simple "cage plugins", FunTraps provides:

- Custom structure-based traps  
- Multi-layer potion systems  
- Region-aware deployment logic  
- Database synchronization  
- Intelligent block restoration  
- Advanced skin system with individual settings  

FunTraps was built with fairness, performance, and modularity in mind.
